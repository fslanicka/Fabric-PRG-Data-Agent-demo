# Eventhouse Data Source Description

> **Copy the content below (between the --- markers) into the "Description" field for your Eventhouse KQL Database data source in the Fabric Data Agent configuration.**

---

This Eventhouse KQL database contains operational data for Prague Václav Havel Airport (PRG) covering January 2024 through December 2025. It includes approximately 5,000 gate assignment records with turnaround tracking, 3,000 crew roster entries mapping crew members to flights, and 2,000 aircraft maintenance event records. Use this data source to answer questions about gate utilization, crew assignments, aircraft maintenance patterns, turnaround efficiency, and operational readiness. This data complements the flight analytics in the Lakehouse data source.
